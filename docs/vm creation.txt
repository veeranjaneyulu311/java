How to Post vm installation settings in Citrix:
-----------------------------------------------

To make internet ready the following things to be done.

1. vi /etc/sysconfig/network-scripts/ifcfg-eth0

	TYPE=Ethernet
	BOOTPROTO=static
	DEFROUTE=yes
	PEERDNS=yes
	PEERROUTES=yes
	IPV4_FAILURE_FATAL=no
	IPV6INIT=yes
	IPV6_AUTOCONF=yes
	IPV6_DEFROUTE=yes
	IPV6_PEERDNS=yes
	IPV6_PEERROUTES=yes
	IPV6_FAILURE_FATAL=no
	IPV6_ADDR_GEN_MODE=stable-privacy
	NAME=eth0
	UUID=6e87d99f-b1b2-4403-bd29-b4e86e957bd4
	DEVICE=eth0
	ONBOOT=yes
	IPADDR=192.168.1.109  // place appropriate static ip for this vm in the local network
	NETMASK=255.255.255.0
	GATEWAY=192.168.1.1

2. Put Sophos client caa file from sophos personal portal and configure according to the instructions given in .caa/README file
	copy caa zip file to root folder and unzip. change username and password and gateway in .caa/caa-conf file
	add /root/bin to path and run caa -d in console.


2. Set up hostname 

	hostnamectl set-hostname elk.dru.care

3. Add The  entry in /etc/hosts file like below  to resolve the  FQDN name with IP address

	cat /etc/hosts
	192.168.1.109 elk.dru.care

Verifying the Hostname details
	hostnamectl status

3. Put the dns records. ( http://ask.xmodulo.com/configure-static-dns-centos-fedora.html )
	
	vi /etc/resolv.conf

		# Generated by NetworkManager
		search elk.dru.care
		nameserver 8.8.8.8
		nameserver 8.8.4.4
